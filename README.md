# Flutter 기차 예매 앱

## 소개
이 프로젝트는 Flutter로 제작한 **기차 예매 애플리케이션**입니다.  
출발역과 도착역을 선택하고, 좌석을 지정한 뒤 예매를 완료할 수 있습니다.

## 사용 기술
- **Dart**
- **Flutter**
- **Material Widgets**
- **Cupertino Widgets**

## ▶사용 방법
1. 출발역과 도착역을 선택합니다.
2. "좌석 선택" 버튼을 클릭합니다.
3. 원하는 좌석을 선택합니다.
4. "예매하기" 버튼을 클릭합니다.
5. 예매 완료 화면을 확인합니다.
